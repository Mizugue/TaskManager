Sobre o projeto:

Este projeto foi desenvolvido utilizando tecnologias modernas como Spring Boot, Hibernate e SQL para criar um sistema simples de gerenciamento de tarefas.

A aplicação permite aos usuários realizar operações básicas de CRUD (Create, Read, Update, Delete), facilitando a administração das tarefas a serem realizadas.

Funcionalidades Principais:
Cadastro de Tarefas: Os usuários podem criar novas tarefas especificando título, descrição, data de vencimento e status.
Atualização de Tarefas: É possível modificar os detalhes das tarefas existentes, como título, descrição, data de vencimento e status atual.
Exclusão de Tarefas: Os usuários têm a opção de remover tarefas que já foram concluídas ou não são mais necessárias.
Consulta de Tarefas: A aplicação permite consultar todas as tarefas cadastradas, seja por meio do ambiente web ou api rest, exibindo detalhes como título, descrição, data de vencimento e status atual.
